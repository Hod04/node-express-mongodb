# A NodeJS setup using Mongoose

## Prerequisites

- npm version 6+
- Node.js version 10+

## .env (for local environment variables)

- Make sure that you have a .env file in the root directory, specifing:
  - PORT
  - NODE_ENV
  - MONGO_URI

## Getting started

- Head over to the root directory and run the following:
  - npm install
  - npm run dev
